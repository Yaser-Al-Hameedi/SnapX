# SnapX  
AI-powered receipts & documents vault.  

## Features  
- Upload receipts & documents  
- Automatic OCR & AI categorization  
- Search by date, type, or amount  
- Dashboard with analytics  

## Tech Stack  
- Frontend: Next.js + Tailwind CSS  
- Backend: FastAPI  
- Database: PostgreSQL  
- Storage: AWS S3  
- AI: AWS Textract + OpenAI
